# Agrobase: A base package for python projects of the AgroReporter environment

This package contains base elements (Connectors, Either, Exceptions, Crops and
Abstract Entities) for the environment building.

## Build a deploy a distribution artifact

For building a distribution package use the wheel builder. The artifact
generated by default `setuptools` package are not supported as an Azure Devops
Artifact. Thus, simple run:

```bash
python setup.py bdist_wheel
```

After build, upload the package to the Agroreporter artifacts feed:

```bash
twine upload \
    --repository-url https://pkgs.dev.azure.com/biotrop/agroreporter/_packaging/agroreporter/pypi/upload \
    src/dist/agrobase-<VERSION NUMBER>-py3-none-any.whl
```

At this moment you will be asked by credentials (the username and the personal
access token - PAT).
